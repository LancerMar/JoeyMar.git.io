C++调用动态链接库（dll）的几种方法

发表于 2019-11-14 | 分类于 C++相关

首先，创建一个新的动态链接库工程，在工程中添加一个.cpp文件，输入以下代码：

点击生成，生成.dll文件，在这里生成的dll名为“Dllfortest.dll”

概述

采用C++编写的应用程序使用dll可以采用两种方式：一种是隐式链接，一种是显式链接。在使用dll前，必须先了解dll的函数结构，打开cmd窗口，输入“dumpbin -exports E:\Dllfortest.dll（文件的绝对路径）”，输出对应可引用的函数名，我这里是“_Sum@8”；另外，Windows系统将遵循下面的搜索顺序来定位DLL： 1．包含EXE文件的目录，2．进程的当前工作目录， 3．Windows系统目录， 4．Windows目录，5．列在Path环境变量中的一系列目录。
新建一个C++控制台工程，引入：

#include <iostream>
#include <Windows.h>

using namespace std;

隐式链接

隐式链接，在程序开始时将动态链接库加载到程序中，使用时将导入函数关键字_declspec(dllimport)写在头文件中即可，将之前生成的dll放在该控制台的可执行程序的同一目录下，引入该函数，获取(5+8)的值，如下所示：
添加头文件Fortest.h,在头文件中输入：

#pragma comment(lib，"Dllfortest.dll")
extern "C"_declspec(dllimport) int Sum(int a,int b);

添加.cpp文件，在文件中输入：

#include "Fortest.h"
int main()
{
    int a;
    a=Sum(5,8);
    cout<<a<<endl;
}

输出结果为：13；
