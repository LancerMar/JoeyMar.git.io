CSharp数据对象转Byte数组

发表于 2019-12-09 | 分类于 CSharp | 阅读次数：

C#程序在与非托管类型的程序通信时（如C/C++），需要将托管类型转化为非托管类型

Marshal

C#在调用非托管代码时常用Marshal方法集合
继承自Object，提供了一个方法集合，这些方法用于分配非托管内存、复制非托管内存块、将托管类型转换为非托管类型，此外还提供了在与非托管代码交互时使用的其他杂项方法
SizeOf(Type)：返回非托管类型的大小（以字节为单位）。

AllocHGlobal(Int32)：通过使用指定的字节数，从进程的非托管内存中分配内存。

StructureToPtr(Object, IntPtr, Boolean)：将数据从托管对象封送到非托管内存块。

  1. Object：包含要封送的数据的托管对象。 该对象必须是格式化类的结构或实例；
  2. IntPtr：指向非托管内存块的指针，必须在调用此方法之前分配该指针；
  3. Boolean：在内存块已包含数据时传递 false 可能会导致内存泄漏；

Copy(IntPtr, byte[], int, int)：将数据从非托管内存指针复制到托管byte数组。

  1. IntPtr：从中进行复制的内存指针；
  2. byte[]：要复制到的数组；
  3. int：目标数组中从零开始的索引，在此处开始复制；
  4. int：要复制的数组元素的数目；

FreeHGlobal(IntPtr)： 释放以前从进程的非托管内存中分配的内存

结构体转化为byte数组

实际应用中，常常需要将结构体序列化为字节流，这里提供在C#与C++相互调用时的一种较为通用的方法。
通过将结构体整体转入非托管内存块，然后拷贝到字节数组。
