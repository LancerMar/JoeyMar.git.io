Linux的C\C++(一)：Linux从安装操作系统到搭建C++环境

发表于 2020-02-14 | 更新于 2020-03-02 | 分类于 Linux | 阅读次数：

最近因为项目原因，需要将一个在Windows上编写的C++的代码迁移到Linux上，开始了解一下Linux的C\C++，记录下踩了的坑。持续学习更新中。。。

Linux安装

官网下载：Ubuntu 然后直接下一步默认操作安装完就行了。
安装C++环境有个坑（后面安装部分会提到），linux并不默认带C++编译环境，需要下载安装，如果直接安装的话，因为apt源是国外的源，安装起来会非常非常慢，修改为国内的源会好很多，这里更改为阿里云源；

Ubuntu更改apt源为阿里云源

  1. 将需要修改的文件 /etc/apt/sources.list 备份，命令：sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
  2. 安装vim，命令：sudo apt-get install vim

  3. 查看当前系统的版本号，命令：lsb_release -c；如下图

    当前版本为 Ubuntu 18.04；
    其他版本对应：
    Ubuntu 12.04 (LTS)代号为precise。
    Ubuntu 14.04 (LTS)代号为trusty。
    Ubuntu 15.04 代号为vivid。
    Ubuntu 15.10 代号为wily。
    Ubuntu 16.04 (LTS)代号为xenial。

  4. 注释或删除原有内容，添加以下内容到list文件中即可

  5. 更新软件列表与更新软件包；命令如下：
    更新软件列表： sudo apt-get update
    更新软件包： sudo apt-get upgrade

C++环境安装

安装命令：sudo apt-get install g++
